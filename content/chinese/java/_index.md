## 常见问题

**问：如何在 Java 中使用 GroupDocs.Merger 合并 PDF？**  
答：使用 `Merger` 类，分别将每个 PDF 加载为 `Document`，然后调用 `merge()` —— API 会自动处理页面顺序和格式一致性。

**问：可以将 PDF 拆分为多个文件吗？**  
答：可以，“split pdf java” 功能允许您定义页面范围或按大小拆分，生成单独的 PDF 文件。

**问：合并后如何添加密码保护？**  
答：在生成的文档上调用 `protect()` 方法，并指定所需的加密级别和密码。

**问：能否将 PDF 与 Excel 表格一起合并？**  
答：完全可以——GroupDocs.Merger 支持跨格式合并，您可以将 PDF 与 Excel 表格（`merge excel sheets java`）合并为单个 PDF 输出。

**问：生产环境需要什么授权？**  
答：部署时需要商业版 GroupDocs.Merger 许可证；我们提供免费试用供评估使用。
