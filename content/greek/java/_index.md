## Πώς να Χωρίσετε PDF με Java;
Διαιρέστε ένα PDF σε μεμονωμένες σελίδες ή προσαρμοσμένα εύρη με μόλις τρεις γραμμές Java. Καλέστε τη μέθοδο `split()` σε μια παρουσία της `Merger`, περάστε τη διαδρομή του πηγαίου αρχείου και καθορίστε το επιθυμητό εύρος σελίδων ή το μέγεθος. Η βιβλιοθήκη γράφει κάθε τμήμα σε ξεχωριστό αρχείο διατηρώντας την αρχική ποιότητα και τα μεταδεδομένα.

Μπορείτε επίσης να διαιρέσετε κατά μέγεθος (π.χ., τμήματα 5 MB) ή με λίστα αριθμών σελίδων, κάτι ιδανικό για τη δημιουργία PDF ανά κεφάλαιο από ένα ενιαίο κύριο έγγραφο. Η λειτουργία εκτελείται σε γραμμικό χρόνο σε σχέση με τον αριθμό των σελίδων, καθιστώντας την κατάλληλη για μεγάλης κλίμακας επεξεργασία παρτίδων.

## Πώς να Συγχωνεύσετε Πολλαπλά PDF με Java;
Φορτώστε κάθε πηγαίο PDF με τη μέθοδο `addDocument()` του `Merger`, τοποθετήστε τα στη ζητούμενη σειρά και καλέστε `merge()`. Το API εναρμονίζει αυτόματα τις διαστάσεις των σελίδων, ενημερώνει τους σελιδοδείκτες και ενοποιεί τις ιδιότητες του εγγράφου. Μπορείτε επίσης να συγχωνεύσετε PDF με άλλες μορφές — όπως Word ή Excel — μετατρέποντάς τα εν κινήσει, παράγοντας ένα ενιαίο, ενοποιημένο PDF.

Για σενάρια υψηλής απόδοσης, ενεργοποιήστε τη λειτουργία streaming ώστε να αποφεύγετε τη φόρτωση ολόκληρων αρχείων στη μνήμη. Αυτό μειώνει τη χρήση heap έως και 80 % κατά την επεξεργασία εγγράφων με εκατοντάδες σελίδες.

## Κατανόηση της Κλάσης Merger
Η κλάση `Merger` είναι το κύριο στοιχείο του GroupDocs.Merger για Java που συντονίζει τις λειτουργίες συνδυασμού, διαίρεσης και ασφάλειας εγγράφων. Εκθέτει ευέλικτες μεθόδους όπως `addDocument()`, `split()`, `protect()` και `merge()` για τη δημιουργία σύντομων αγωγών επεξεργασίας.

### Επισκόπηση Κύριων Μεθόδων
- `addDocument(String path)`: Προσθέτει ένα πηγαίο αρχείο στη σειρά για μελλοντική συγχώνευση.  
- `split(String source, SplitOptions options)`: Διαχωρίζει ένα έγγραφο βάσει εύρους σελίδων ή ορίων μεγέθους.  
- `protect(String output, ProtectionOptions options)`: Εφαρμόζει προστασία κωδικού πρόσβασης και περιορισμούς δικαιωμάτων.  
- `merge(String output)`: Εκτελεί τις προγραμματισμένες λειτουργίες και γράφει το τελικό έγγραφο.  

Αυτές οι μέθοδοι είναι thread‑safe και μπορούν να αλυσοδεθούν για εκφραστικό, αναγνώσιμο κώδικα.

## Συχνές Ερωτήσεις
**Q: Πώς μπορώ να χωρίσω PDF χρησιμοποιώντας το GroupDocs.Merger σε Java;**  
A: Δημιουργήστε μια παρουσία `Merger`, καλέστε `split(sourcePath, new SplitOptions().setPageRanges("1-3,5,7-10"))` και καθορίστε έναν φάκελο εξόδου — κάθε εύρος γίνεται ξεχωριστό αρχείο PDF.

**Q: Μπορώ να συγχωνεύσω PDF και φύλλα Excel μαζί;**  
A: Ναι — το GroupDocs.Merger υποστηρίζει διαμορφο-διασυγχώνευση, επιτρέποντάς σας να συνδυάσετε PDF με αρχεία Excel (`merge excel files java`) σε μια ενιαία έξοδο PDF.

**Q: Πώς μπορώ να προσθέσω προστασία κωδικού μετά τη συγχώνευση;**  
A: Μετά την κλήση του `merge()`, καλέστε `protect(outputPath, new ProtectionOptions().setPassword("Secret123"))` για κρυπτογράφηση του τελικού εγγράφου.

**Q: Είναι δυνατόν να συγχωνεύσετε PDF χωρίς να φορτώνετε ολόκληρο το αρχείο στη μνήμη;**  
A: Ενεργοποιήστε το streaming (`Merger.setStreaming(true)`) για επεξεργασία αρχείων με ενδιάμεση αποθήκευση, μειώνοντας δραστικά την κατανάλωση μνήμης για μεγάλα έγγραφα.

**Q: Ποια αδειοδότηση απαιτείται για παραγωγική χρήση;**  
A: Μια εμπορική άδεια GroupDocs.Merger είναι υποχρεωτική για παραγωγικές εγκαταστάσεις· μια δωρεάν δοκιμή 30 ημερών είναι διαθέσιμη για ανάπτυξη και δοκιμές.
