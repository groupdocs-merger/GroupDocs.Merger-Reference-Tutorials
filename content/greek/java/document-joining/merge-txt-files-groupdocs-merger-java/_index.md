## Οδηγός Υλοποίησης

### Συγχώνευση Πολλαπλών Αρχείων TXT

#### Επισκόπηση
Παρακάτω υπάρχει ένας βήμα‑βήμα οδηγός που δείχνει **πώς να συγχωνεύσετε πολλαπλά αρχεία txt** χρησιμοποιώντας το GroupDocs.Merger για Java. Το μοτίβο κλιμακώνεται από δύο αρχεία σε δεκάδες χωρίς αλλαγές κώδικα.

#### Βήμα 1: Φόρτωση Πηγών Αρχείων
Πρώτα, ορίστε τις διαδρομές των αρχείων που θέλετε να συνδυάσετε και δημιουργήστε ένα αντικείμενο `Merger` για το αρχικό αρχείο:

```java
import com.groupdocs.merger.Merger;

String sourceFilePath1 = "YOUR_DOCUMENT_DIRECTORY/sample1.txt";
String sourceFilePath2 = "YOUR_DOCUMENT_DIRECTORY/sample2.txt";

Merger merger = new Merger(sourceFilePath1);
```

#### Βήμα 2: Προσθήκη Επιπλέον Αρχείων
Χρησιμοποιήστε τη μέθοδο `join` για να προσαρτήσετε κάθε επόμενο αρχείο TXT στο βασικό έγγραφο. Μπορείτε να καλέσετε `join` όσες φορές χρειάζεται—ιδανικό για σενάρια **συγχώνευσης πολλαπλών txt**:

```java
merger.join(sourceFilePath2); // Merge second TXT file into the first one
```

#### Βήμα 3: Αποθήκευση Συγχωνευμένου Αποτελέσματος
Τέλος, γράψτε το συνδυασμένο περιεχόμενο σε μια νέα τοποθεσία αρχείου:

```java
String outputFilePath = "YOUR_OUTPUT_DIRECTORY/merged.txt";
merger.save(outputFilePath);
```

- **Διαχείριση Μνήμης:** Όταν συγχωνεύετε πολύ μεγάλα αρχεία, σκεφτείτε να τα επεξεργάζεστε σε παρτίδες και παρακολουθήστε τη μνήμη heap της JVM για να αποφύγετε `OutOfMemoryError`.
