## Πώς να συγχωνεύσετε αρχεία PPTX με το GroupDocs.Merger

### Φόρτωση Πηγαίου Αρχείου
**Βήμα 1 – Καθορίστε τη διαδρομή του εγγράφου**

```java
String filePath = "YOUR_DOCUMENT_DIRECTORY/sample.pptx";
```

Βεβαιωθείτε ότι η διαδρομή δείχνει σε ένα υπάρχον αρχείο PPTX· διαφορετικά θα ριχτεί `FileNotFoundException`.

**Βήμα 2 – Αρχικοποιήστε το αντικείμενο Merger**

```java
Merger merger = new Merger(filePath);
```

Η παρουσία `Merger` τώρα αντιπροσωπεύει την πρώτη παρουσίαση με την οποία θέλετε να εργαστείτε.

### Πώς να ενώνετε αρχεία PPTX προγραμματιστικά
**Βήμα 1 – Ορίστε τις πρόσθετες διαδρομές αρχείων**

```java
String filePath1 = "YOUR_DOCUMENT_DIRECTORY/sample.pptx";
String filePath2 = "YOUR_DOCUMENT_DIRECTORY/additional_sample.pptx";
```

`filePath1` είναι η κύρια παρουσίαση· `filePath2` (και τυχόν περαιτέρω αρχεία) θα προσαρτηθούν.

**Βήμα 2 – Φορτώστε το κύριο αρχείο**

```java
Merger merger = new Merger(filePath1);
```

**Βήμα 3 – Προσθέστε τις επιπλέον παρουσιάσεις**

```java
merger.join(filePath2);
```

Μπορείτε να καλέσετε το `join` επανειλημμένα για να συνδυάσετε τρεις, τέσσερις ή περισσότερες παρουσιάσεις.

**Βήμα 4 – Αποθηκεύστε το συγχωνευμένο αποτέλεσμα**

```java
String outputFile = "YOUR_OUTPUT_DIRECTORY/merged_output.pptx";
merger.save(outputFile);
```

Μετά από αυτήν την κλήση θα βρείτε ένα ενιαίο αρχείο PPTX που περιέχει όλες τις διαφάνειες από τα πηγαία αρχεία.

## Συχνές Ερωτήσεις

**Q: Ποια άλλα φορμάτ μπορεί να διαχειριστεί το GroupDocs.Merger;**  
A: Εκτός από PPTX, η βιβλιοθήκη υποστηρίζει PDF, DOCX, XLSX και πολλά άλλα είδη εγγράφων.

**Q: Υπάρχει τρόπος να προστατέψετε τη συγχωνευμένη παρουσίαση με κωδικό πρόσβασης;**  
A: Ναι – μετά τη συγχώνευση, μπορείτε να καλέσετε `merger.protect("password")` για να προσθέσετε κρυπτογράφηση.

**Q: Μπορώ να συγχωνεύσω παρουσιάσεις αποθηκευμένες σε αποθήκευση cloud (π.χ., AWS S3);**  
A: Απόλυτα. Φορτώστε τα αρχεία σε ένα `byte[]` ή `InputStream` και περάστε τα στον κατασκευαστή `Merger`.

**Q: Διατηρεί η βιβλιοθήκη τις κινήσεις και τις μεταβάσεις;**  
A: Όλες οι εγγενείς λειτουργίες του PowerPoint, συμπεριλαμβανομένων των animations, transitions και slide masters, διατηρούνται κατά τη συγχώνευση.

**Q: Πώς μπορώ να συγχωνεύσω περισσότερα από δύο αρχεία PPTX με μία κλήση;**  
A: Προετοιμάστε μια `List<String>` με τις διαδρομές αρχείων και επαναλάβετε `merger.join(path)` για κάθε στοιχείο.
