## java συγχώνευση αρχείων excel – Οδηγός Βήμα‑Βήμα

### Αρχικοποίηση Αντικειμένου Merger
**Επισκόπηση:**  
Για να ξεκινήσετε τη συγχώνευση, χρειάζεστε πρώτα ένα αρχικοποιημένο αντικείμενο Merger. Αυτό το βήμα προετοιμάζει το περιβάλλον για λειτουργίες αρχείων.

#### Βήμα 1: Ορισμός Διαδρομής Πηγής Εγγράφου
Ορίστε τη διαδρομή όπου βρίσκεται το κύριο αρχείο Excel σας.
```java
String sourceFilePath = "YOUR_DOCUMENT_DIRECTORY/sample.xlsx";
```

#### Βήμα 2: Δημιουργία Αντικειμένου Merger
Αρχικοποιήστε το με το καθορισμένο αρχείο πηγής.
```java
Merger merger = new Merger(sourceFilePath);
```
Αυτό το αντικείμενο λειτουργεί ως βάση για όλες τις επόμενες λειτουργίες.

### Συγχώνευση Πολλαπλών Αρχείων XLSX
**Επισκόπηση:**  
Η συγχώνευση πολλαπλών αρχείων σε ένα είναι απλή. Ας δούμε πώς μπορείτε να συνδυάσετε πολλά έγγραφα Excel.

#### Βήμα 1: Ορισμός Καταλόγου Εξόδου και Διαδρομής Αρχείου
Καθορίστε πού θα αποθηκευτεί το συγχωνευμένο αρχείο.
```java
String outputFilePath = "YOUR_OUTPUT_DIRECTORY/merged.xlsx";
```

#### Βήμα 2: Αρχικοποίηση Merger με το Πρώτο Αρχείο XLSX
Ξεκινήστε φορτώνοντας το κύριο έγγραφο.
```java
Merger merger = new Merger("YOUR_DOCUMENT_DIRECTORY/sample.xlsx");
```

#### Βήμα 3: Προσθήκη Άλλου Αρχείου XLSX για Συγχώνευση
Χρησιμοποιήστε τη μέθοδο `join` για την προσθήκη περισσότερων αρχείων.
```java
merger.join("YOUR_DOCUMENT_DIRECTORY/sample2.xlsx");
```
Μπορείτε να επαναλάβετε αυτήν την κλήση `join` για όσα λογιστικά φύλλα χρειάζεστε να **συγχωνεύσετε πολλαπλά αρχεία xlsx**.

#### Βήμα 4: Αποθήκευση του Συγχωνευμένου Αποτελέσματος
Τέλος, αποθηκεύστε τη δουλειά σας.
```java
merger.save(outputFilePath);
```

**Συμβουλές Επίλυσης Προβλημάτων:**  
- Βεβαιωθείτε ότι όλες οι διαδρομές αρχείων είναι σωστές για να αποφύγετε το `FileNotFoundException`.  
- Επαληθεύστε τα δικαιώματα εγγραφής για τον κατάλογο εξόδου.  
- Για μεγάλα βιβλία εργασίας, σκεφτείτε την επεξεργασία τους σε μικρότερες παρτίδες για βελτίωση της χρήσης μνήμης.
