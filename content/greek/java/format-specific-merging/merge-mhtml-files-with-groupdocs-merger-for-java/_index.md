## Οδηγός Υλοποίησης

### Φόρτωση και Συγχώνευση Αρχείων MHTML

#### Επισκόπηση

Ο συνδυασμός αρχείων MHTML απλοποιεί τη διαδικασία διαχείρισης περιεχομένου βασισμένου στο web, καθιστώντας πιο εύκολη τη διανομή ή την αρχειοθέτηση. Με το GroupDocs.Merger, αυτή η εργασία γίνεται αβίαστη.

#### Οδηγίες Βήμα‑Βήμα

**1. Ορίστε τον Κατάλογο Εξόδου**  

Καθορίστε πού θέλετε να αποθηκευτεί το συγχωνευμένο αρχείο:
```java
String outputFolder = "YOUR_OUTPUT_DIRECTORY";
```

**2. Αρχικοποιήστε το Merger με το Πρώτο Αρχείο MHTML**  

Φορτώστε το αρχικό αρχείο πηγής για να ξεκινήσετε τη συγχώνευση:
```java
Merger merger = new Merger("YOUR_DOCUMENT_DIRECTORY/SAMPLE_MHTML");
```

*Εξήγηση:* Το `Merger` αρχικοποιείται με τη διαδρομή του πρώτου εγγράφου MHTML.

**3. Προσθέστε Επιπλέον Αρχεία MHTML**  

Προσθέστε περισσότερα αρχεία χρησιμοποιώντας τη μέθοδο `join`:
```java
merger.join("YOUR_DOCUMENT_DIRECTORY/SAMPLE_MHTML_2");
```

*Εξήγηση:* Αυτό το βήμα προσθέτει ένα ακόμη αρχείο MHTML στην παρουσία του merger, προετοιμάζοντάς το για ενιαία έξοδο.

**4. Αποθηκεύστε το Συγχωνευμένο Αποτέλεσμα**  

Τέλος, γράψτε το συνδυασμένο έγγραφο στο δίσκο:
```java
String outputFile = new File(outputFolder, "merged.mhtml").getPath();
merger.save(outputFile);
```

*Εξήγηση:* Η μέθοδος `save` ενοποιεί όλα τα προστιθέμενα αρχεία σε ένα που καθορίζεται από το `outputFile`.
