## Οδηγός Υλοποίησης

Ας αναλύσουμε κάθε λειτουργία συγχώνευσης PDF με τη GroupDocs.Merger for Java σε διαχειρίσιμα τμήματα.

### Φόρτωση Πηγαίου PDF

#### Επισκόπηση
Αυτή η ενότητα δείχνει πώς να φορτώσετε ένα πηγαίο αρχείο PDF, το οποίο είναι το πρώτο βήμα πριν εκτελεστεί οποιαδήποτε λειτουργία συγχώνευσης.

**Βήματα:**
1. **Ορίστε τον φάκελο εγγράφων**: Ρυθμίστε τη διαδρομή όπου αποθηκεύονται τα αρχεία PDF σας.  
2. **Αρχικοποιήστε το αντικείμενο Merger**: Χρησιμοποιήστε αυτό το αντικείμενο για να διαχειριστείτε το φορτωμένο αρχείο PDF.

```java
import com.groupdocs.merger.Merger;

String documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
Merger merger = new Merger(documentDirectory + "/sample_pdf_1.pdf");
```

### Προσθήκη Άλλου PDF για Συγχώνευση

#### Επισκόπηση
Αφού φορτώσετε ένα πηγαίο PDF, μπορείτε να προσθέσετε επιπλέον αρχεία PDF για συγχώνευση.

**Βήματα:**
1. **Επανααρχικοποίηση ή επαναχρησιμοποίηση του υπάρχοντος αντικειμένου Merger**: Βεβαιωθείτε ότι δείχνει στο αρχικό PDF σας.  
2. **Προσθήκη άλλου αρχείου PDF**: Χρησιμοποιήστε τη μέθοδο `join` για να συμπεριλάβετε επιπλέον έγγραφα στη διαδικασία συγχώνευσης.

```java
import com.groupdocs.merger.Merger;

String documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
Merger merger = new Merger(documentDirectory + "/sample_pdf_1.pdf");
merger.join(documentDirectory + "/sample_pdf_2.pdf");
```

### Αποθήκευση Συγχωνευμένου Αποτελέσματος

#### Επισκόπηση
Το τελικό βήμα είναι η αποθήκευση του συγχωνευμένου PDF σε έναν καθορισμένο φάκελο εξόδου.

**Βήματα:**
1. **Ορίστε διαδρομές για τους φακέλους εγγράφων και εξόδου**: Καθορίστε πού θέλετε να βρίσκονται τα αρχεία εισόδου και το παραγόμενο αρχείο.  
2. **Αποθηκεύστε το συγχωνευμένο έγγραφο**: Χρησιμοποιήστε τη μέθοδο `save` για να αποθηκεύσετε το συνδυασμένο PDF στην επιθυμητή τοποθεσία.

```java
import com.groupdocs.merger.Merger;

String documentDirectory = "YOUR_DOCUMENT_DIRECTORY";
String outputDirectory = "YOUR_OUTPUT_DIRECTORY";

Merger merger = new Merger(documentDirectory + "/sample_pdf_1.pdf");
merger.join(documentDirectory + "/sample_pdf_2.pdf");

String outputFile = outputDirectory + "/merged_output.pdf";
merger.save(outputFile);
```
