- **Mesclagem abrangente de documentos** – Combine documentos inteiros ou páginas específicas de várias fontes em um único arquivo coeso.

- **Divisão inteligente de documentos** – Separe documentos por número de página, intervalos ou extraia páginas específicas como arquivos separados.

- **Gerenciamento de segurança** – Adicione, remova ou atualize senhas e permissões de documentos.

- **Sem dependências externas** – Processa documentos sem exigir Microsoft Office ou software Adobe.  
- **Alto desempenho** – Otimizada para processar documentos grandes e operações em lote de forma eficiente.

## Perguntas Frequentes

**Q: Posso usar a API em um produto comercial?**  
A: Sim, com uma licença válida da GroupDocs. Um teste gratuito está disponível para avaliação.

**Q: A API suporta PDFs protegidos por senha?**  
A: Absolutamente. Você pode abrir, modificar e re‑proteger PDFs criptografados usando as mesmas chamadas da API.

**Q: Quais versões do .NET e Java são compatíveis?**  
A: .NET Standard 2.0, .NET Framework 2.0+, .NET Core 2.0+, Mono 1.2+; Java 7, 8 e 10+ são totalmente suportados.

**Q: Como a API lida com arquivos muito grandes?**  
A: Ela faz streaming dos dados e utiliza algoritmos de baixo consumo de memória, permitindo processar documentos de vários gigabytes sem esgotar os recursos do sistema.

**Q: Existe suporte integrado para OCR ou extração de texto?**  
A: Embora a API central de mesclagem foque em operações estruturais, você pode combiná‑la com o GroupDocs.Parser para extração avançada de texto e necessidades de OCR.

**Q: Posso mesclar diferentes tipos de arquivo em um único PDF?**
