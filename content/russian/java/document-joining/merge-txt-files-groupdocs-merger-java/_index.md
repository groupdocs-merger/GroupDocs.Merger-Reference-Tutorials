## Руководство по реализации

### Объединение нескольких TXT‑файлов

#### Обзор
Ниже представлена пошаговая инструкция, показывающая **как объединять несколько txt** файлов с помощью GroupDocs.Merger for Java. Шаблон масштабируется от двух файлов до десятков без изменения кода.

#### Шаг 1: Загрузка исходных файлов
Сначала задайте пути к файлам, которые хотите объединить, и создайте объект `Merger` для начального файла:

```java
import com.groupdocs.merger.Merger;

String sourceFilePath1 = "YOUR_DOCUMENT_DIRECTORY/sample1.txt";
String sourceFilePath2 = "YOUR_DOCUMENT_DIRECTORY/sample2.txt";

Merger merger = new Merger(sourceFilePath1);
```

#### Шаг 2: Добавление дополнительных файлов
Используйте метод `join`, чтобы добавить каждый последующий TXT‑файл к базовому документу. Вы можете вызывать `join` столько раз, сколько необходимо — идеально для сценариев **merge multiple txt**:

```java
merger.join(sourceFilePath2); // Merge second TXT file into the first one
```

#### Шаг 3: Сохранение объединённого результата
Наконец, запишите объединённое содержимое в новое место назначения:

```java
String outputFilePath = "YOUR_OUTPUT_DIRECTORY/merged.txt";
merger.save(outputFilePath);
```

### Советы по устранению неполадок
- **Проблемы с путями к файлам:** Убедитесь, что каждый путь абсолютный или правильно относительный к рабочему каталогу.  
- **Управление памятью:** При объединении очень больших файлов рассматривайте их обработку пакетами и следите за кучей JVM, чтобы избежать `OutOfMemoryError`.

| **OutOfMemoryError** | Обрабатывайте файлы небольшими партиями или увеличьте размер кучи JVM. |
