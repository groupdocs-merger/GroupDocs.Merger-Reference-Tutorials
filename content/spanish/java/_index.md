## Visión General Rápida

- **Capacidad principal:** Split PDF Java documents into custom page ranges.  
- **Poderes adicionales:** How to merge PDF files, convert Word to PDF, merge images into PDF, and secure PDFs with passwords.

- **Fusionar varios documentos** into a single file while preserving formatting and content integrity.

## Cómo Dividir PDF Java con GroupDocs.Merger

Dividir un PDF es a menudo el primer paso cuando necesitas aislar un capítulo, extraer una sola factura o preparar archivos para procesamiento posterior. Con GroupDocs.Merger puedes:

1. **Cargar el PDF de origen** – la API acepta archivos, flujos o URLs.  
2. **Definir los rangos de páginas** que deseas conservar o descartar.  
3. **Guardar cada rango como un nuevo PDF** – el resultado es un conjunto de documentos limpios e independientes listos para su distribución.
